# コンセプト

## アプリの概要

このアプリは、Notion の直感的なインターフェースと Obsidian の柔軟性を融合させた次世代のナレッジマネジメントツールです。日常のメモから複雑な知識体系の構築、タスク管理まで、あらゆる情報整理のニーズに応える統合プラットフォームを目指しています。

私たちの目標は、「思考をシームレスにデジタル化する」ことです。ユーザーが考えたことをすぐに記録し、整理し、必要なときに即座に取り出せる環境を提供します。

### メモ帳

日常の記録や、情報の整理、日記などを書くのに最適な環境を提供します。

- **瞬時の起動と記録**: アプリを開いてすぐに書き始められる、ストレスフリーな体験
- **リッチテキスト編集**: テキスト、画像、表、コードブロックなど多様なコンテンツをサポート
- **自動保存**: 入力内容は自動的に保存され、データ損失の心配がない
- **タグ付け**: メモに複数のタグを付けることで、後から様々な視点で情報を整理・検索可能
- **テンプレート**: 日記、会議メモ、レシピなど、用途に応じたテンプレートを提供

#### ユースケース例

- 会議中の素早いメモ取り
- 日々の思考や気づきの記録
- 読書ノートの作成
- レシピや手順書の保存
- 旅行の計画や記録

### パーソナルナレッジベース

蓄積した情報にすぐにアクセスできるように、充実した検索機能、ツリー表示、データベース表示をサポートします。知識の関連性を視覚化し、新たな気づきを促進します。

- **強力な検索機能**: 全文検索、タグ検索、プロパティ検索など多様な検索方法
- **双方向リンク**: ノート間の関連性を双方向リンクで表現し、知識のネットワークを構築
- **知識グラフ**: ノート間の関連性を視覚的に表示するグラフビュー
- **データベースビュー**: 表形式、カンバン、ギャラリーなど多様な表示形式
- **埋め込み表示**: 他のノートの内容を現在のノートに埋め込み表示する機能

#### ユースケース例

- 研究プロジェクトの情報管理
- 学習内容の体系化
- 趣味や専門分野の知識ベース構築
- 仕事のナレッジベース
- アイデアの発展と関連付け

### タスク管理

ノートをタスクとしてタグ付けすることで、シームレスなタスク管理を実現します。知識ベースとタスク管理を統合することで、コンテキストを失うことなく行動に移せます。

- **タスクの視覚化**: カンバンボード、リスト、カレンダーなど多様な表示形式
- **優先度と期限**: タスクの重要度と期限を設定し、視覚的に表示
- **サブタスク**: 複雑なタスクを小さな単位に分割して管理
- **進捗追跡**: タスクの完了状況をリアルタイムで追跡
- **リマインダー**: 重要なタスクの期限が近づくとリマインダーを表示

#### ユースケース例

- プロジェクト管理
- 日常のToDo管理
- 習慣トラッキング
- 目標達成のためのマイルストーン設定
- チームでのタスク共有（サーバー機能使用時）

## アプリの特徴

### オープンソース

このアプリは完全にオープンソースで開発され、コミュニティによる継続的な改善と拡張を可能にします。

- **透明性**: すべてのコードがオープンで、セキュリティや機能を誰でも検証可能
- **コミュニティ貢献**: バグ修正や機能追加をコミュニティメンバーが貢献可能
- **永続性**: 特定の企業に依存せず、長期的な存続が保証される
- **自由なカスタマイズ**: 必要に応じて独自の機能を追加・変更可能
- **無料利用**: 商用利用を含め、完全に無料で利用可能

### WYSIWYG

- このアプリはNotionと同様の直感的な編集体験を提供します。
    - ユーザーはMarkdownを意識する必要はなく、見たままの編集が可能
    - マウス操作による直感的な要素の選択や編集ができる
    - スラッシュコマンド（/）で様々な要素を素早く挿入
    - ドラッグ＆ドロップによる画像挿入や要素の並べ替え
- ノートの保存形式はMarkdownまたはその拡張形式とし、長期的な可読性と互換性を確保します。
- WYSIWYGで編集するモードと、生のMarkdownを直接編集するモードを切り替え可能です。
    - 高度なユーザーは必要に応じてMarkdownを直接編集できる
    - 両モード間でシームレスに切り替え可能
    - Markdownの知識がなくても使いこなせる設計

### 階層のないノート

- すべてのノートはストレージ上ではフラットな構造で保存されます。
    - ファイル数の上限が問題とならないように、作成された日付でディレクトリは分けてノートが保存される
    - 例: `notes/2025/03/01/note-title-uuid.md`
- ノート間の親子関係はMarkdownのメタデータ（プロパティ）として定義されます
    - 物理的なファイル構造に縛られない柔軟な整理が可能
    - 一つのノートが複数のコンテキストに属することも可能
- 親子関係によるツリー表示や、検索によってノートが参照されます
    - 様々な視点からノートを整理・閲覧可能
    - 同じノートを複数の場所から参照可能
- ディレクトリ構造がない分、検索やツリー、データベースの機能が充実しています
    - フルテキスト検索、タグ検索、プロパティ検索など多様な検索方法
    - カスタマイズ可能なデータベースビュー
    - 動的なフィルタリングとソート
- ノートを書き始めるときにどこに保存するか考える必要がありません
    - 「書く」と「整理する」の作業を分離し、思考の流れを妨げない
    - 後からタグ付けやプロパティ設定で整理可能
    - 検索性能の高さにより、物理的な整理の必要性が低減

### エディタとしての機能

- VS Codeのようなタブ、分割表示をサポートし、複数のノートを同時に編集・参照できます
    - ドラッグ＆ドロップによる直感的なウィンドウ分割
    - タブのグループ化と整理
    - 画面分割によるノート間の比較や参照
- アウトライン表示により、長文ノートの構造を把握し、素早く移動できます
    - 見出しレベルに応じた階層表示
    - クリックによる該当箇所への即時ジャンプ
    - 折りたたみ機能による概要の把握
- ノートの最後に表示した位置やカーソルを記憶し、次回開いたときに同じ場所から編集を再開できます
    - 長文の編集でも作業の連続性を保証
    - デバイス間での編集位置の同期（サーバー機能使用時）
    - 編集履歴の追跡と復元

### アプリとしての機能

- オフラインでもローカルのストレージで完結して編集でき、インターネット接続に依存しません
    - すべてのデータはローカルに保存され、プライバシーを確保
    - オフライン環境でも全機能が利用可能
    - 同期はオプションであり、必須ではない
- プラグインをサポートし、コミュニティによる機能拡張を促進します
    - アプリの機能の多くがプラグインとして実装され、必要な機能だけを選択可能
    - JavaScript/TypeScriptによる簡単なプラグイン開発
    - プラグインマーケットプレイスによる共有と発見
    - APIによるサードパーティツールとの連携
- テーマをサポートし、ユーザー好みの外観にカスタマイズできます
    - ライト/ダークモードの切り替え
    - カスタムCSSによる細かな調整
    - コミュニティ作成のテーマの共有
    - フォントやカラースキームのカスタマイズ

### サーバー

- 複数端末で同期編集を可能とするオプションのサーバー機能を提供します
    - ユーザーはこれを使わなくても良いが、使用することで以下の機能が利用可能になります
    - リアルタイム同期: 複数デバイス間でのシームレスな同期
    - バックアップ: クラウドへの自動バックアップによるデータ保護
    - インターネット公開: 特定のノートを公開ウェブページとして共有
    - コラボレーション: 複数ユーザーでのリアルタイム共同編集
    - バージョン履歴: 変更履歴の保存と復元
- セルフホスティングオプションにより、プライバシーとデータの所有権を確保できます
    - 自前のサーバーでホスティング可能
    - Docker対応で簡単デプロイ
    - クラウドプロバイダーへの簡単なデプロイガイド提供

### サポートOS

以下のOSでネイティブアプリとして動作し、デバイスを問わず一貫した体験を提供します：

- **Linux**: 主要ディストリビューションに対応
  - Debian/Ubuntu系
  - Red Hat/Fedora系
  - Arch Linux系
- **Windows**: Windows 10以降をサポート
  - Microsoft Storeからのインストールオプション
  - ポータブル版も提供
- **macOS**: macOS 11 Big Sur以降に対応
  - Apple Silicon最適化
  - Mac App Storeからのインストールオプション
- **Android**: Android 8.0以降をサポート
  - タブレット向け最適化UI
  - オフライン機能の完全サポート
- **iOS/iPadOS**: 将来的なサポートを計画
  - 開発ロードマップに含まれる予定

## ユーザー体験の原則

### シンプルさと直感性

- 初心者でも迷わず使えるシンプルなUI
- 必要な機能にすぐにアクセスできる設計
- 学習曲線を最小限に抑えた操作性

### 柔軟性と拡張性

- ユーザーの使い方に合わせて成長するシステム
- 基本機能は簡潔に、高度な機能は必要に応じて拡張
- プラグインによる無限の可能性

### パフォーマンスとレスポンシブ性

- 大量のノートでも快適に動作する高速なパフォーマンス
- 入力遅延のないレスポンシブな編集体験
- リソース使用の最適化

### プライバシーとデータ所有権

- ユーザーデータは常にユーザーのコントロール下に
- 暗号化によるデータ保護
- オープンなデータ形式による長期的なアクセス保証

## コミュニティとエコシステム

このアプリは単なるソフトウェアを超え、知識管理と生産性向上に情熱を持つコミュニティの中心となることを目指します。

- **オープンな開発プロセス**: 開発ロードマップの公開と優先順位付けへのコミュニティ参加
- **コントリビューターの多様性**: 技術的貢献だけでなく、翻訳、ドキュメント、デザイン、アイデア提供など様々な形での参加
- **学習リソース**: チュートリアル、ガイド、ベストプラクティスの共有
- **ユースケースの共有**: 様々な分野や目的でのアプリ活用法の共有
- **プラグインとテーマのエコシステム**: コミュニティによる拡張機能の開発と共有

## ビジョン

このアプリは、単なるメモツールを超え、ユーザーの第二の脳として機能することを目指します。思考を整理し、知識を構造化し、アイデアを発展させる知的活動のパートナーとなります。

私たちは、テクノロジーが人間の創造性と生産性を拡張する可能性を信じています。このアプリを通じて、ユーザーが情報の海に溺れることなく、自分の知識を効果的に管理し、新たな洞察を得られるよう支援します。

オープンソースの精神に基づき、このツールが多くの人々の知的活動を支え、個人の成長だけでなく、集合知の発展にも貢献することを願っています。
